VM Overview

The VM Overview page displays information for the selected VM. The Overview page displays the following portlets that provide information about the VM.

  • Storage

  • Software/Firmware

  • News Feed

For detailed information about portlet items, do one of the following:

  • Click More at the bottom of the portlet

  • Select the name of the portlet from the navigation list.

For more information on the controls and operation of the Overview page, see HP Insight Management pages

Storage information

If the Storage Module for vCenter is installed, the Overview page includes the following:

  • A Storage portlet

  • Array health status information

  • Array firmware version information

  • Storage-related News Feed topics

For more information, see Navigating HP Insight Management with the Storage Module.
