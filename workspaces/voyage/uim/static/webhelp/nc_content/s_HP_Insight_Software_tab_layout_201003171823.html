Navigating HP Insight Management with the Storage Module

HP Insight Management displays information about HP storage systems. For more information, see Accessing Insight Control for vCenter.

The Overview page for servers and clusters is shared between the Server Module and Storage Module. The Overview page for datastores and VMs is available only when the Storage Module is installed. For more information, see the following:

  • Managing clusters

  • Managing datastores

  • Managing VMs
