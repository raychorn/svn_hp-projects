Create Virtual Machines – Specify VMs

Define the properties for the virtual machine being created.

  1. Select the host or cluster that will own the new VMs in Owner.

    This information is entered automatically if you initiated the VM creation from a host or cluster.

  2. Select the template to use for the new VMs in VM Template.

    This information is entered automatically if you initiated the VM creation from a VM template.

  3. Enter the number of VMs to create in VM Quantity.

  4. Specify whether the initial VM state is Power On or Power Off in VM Initial State.

  5. Specify a customization specification (if available) in Customization Specification.

    For more information, see the VMware documentation.

    NOTE: Check the Per VM and Total capacity required for the new VM(s) information displayed in the bottom half of the window. You may need this information in the next step.

  6. Click Next.
