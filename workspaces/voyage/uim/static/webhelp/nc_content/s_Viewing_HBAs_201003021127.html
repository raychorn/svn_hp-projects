Viewing HBAs

The HBAs page displays the following information:

  • Host Names—Host name or cluster host names

  • Type—HBA type

  • VMware Device Identifier—HBA identifier assigned by VMware

  • WWN/IQN—HBA WWN

    NOTE: HBA WWN address is not available for HP MSA P2000 G3 SAS arrays.

  • Label—HBA identifier assigned by VMware
