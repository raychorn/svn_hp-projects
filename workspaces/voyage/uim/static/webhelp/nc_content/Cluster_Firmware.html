Cluster software/firmware

This page provides all software /firmware details of the host added under the cluster and also displays the status of firmware updates.

Details view

This view identifies the firmware and software components installed on hosts in the cluster.

  • Host—The host in the cluster.

  • Type—The type of component (software or firmware).

  • Name—The component on which the firmware or software is installed.

  • Description—The type of firmware or software.

  • Version—The firmware or software version installed.

Status Update view

This view includes the following job-related update information.

  • Server—The host on which the job runs.

  • Job Description—A description of any current or pending jobs for the server.

  • Status—The status of the job.

  • Percent Complete—Indicates how much of the job has completed.

TIP: To update firmware, select Firmware from the Actions drop-down list.
