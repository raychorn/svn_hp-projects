Firewall port requirements

To achieve full functionality, an Insight Control for vCenter server requires that the firewall ports selected or configured during the installation of Insight Control for vCenter 7.0 server are opened (inbound/outbound). The system on which the software package is installed needs to route via a firewall to Onboard Administrators/ Virtual Connects.

Insight Control for vCenter uses outbound pings to determine if entities are available. Therefore, ICMP ping must be enabled.
