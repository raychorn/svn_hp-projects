Create Datastores – Specify Storage

Select the number of datastores to create and their storage properties.

  1. Enter the number of datastores to create in Quantity.

  2. Enter the capacity for the new datastores in Datastore Capacity.

    NOTE: The actual value used by the software might be slightly higher than specified in order to accommodate the VMware overhead.

  3. Select a storage pool to use for the new datastore in Storage Pools.

    The configured storage pools with Full Access on arrays that support provisioning are listed.

    TIP: Select the Name or Available Capacity option to sort the list of storage pools.

    The Capacity Before and Capacity After values for the available storage pools is displayed.

    If there is not enough space available in the selected storage pool, a message is displayed in the Warnings box.

  4. Select a RAID level for the new datastores in RAID Level.

    All RAID levels supported by the selected storage pool are listed.

  5. Click Next.
