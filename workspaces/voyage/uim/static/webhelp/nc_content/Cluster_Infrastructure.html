Cluster infrastructure information

To view infrastructure information about a selected cluster:

  1. Select a cluster in the inventory tree.

  2. Click the HP Insight Management tab.

  3. Select Infrastructure.

    The Infrastructure page opens listing details about the enclosure, power, and thermal status.

    TIP: Use the expand icon at the beginning of each row to see details for the component.
