Network portlet for blades is missing the diagram link

Possible cause: Virtual Connect credentials are missing or incorrect.

Action: Use the Host Properties page or the Server Password Management page to verify that the Virtual Connect credentials are correct and reachable for Insight Control for vCenter, and change them if necessary.
