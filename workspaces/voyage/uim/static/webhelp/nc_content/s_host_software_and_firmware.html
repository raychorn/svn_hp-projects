Viewing host software and firmware

The host software and firmware page displays detailed information about the host software and firmware configuration. The Software/Firmware portlet, which displays an overview of the software and firmware, is displayed on the Overview page.

NOTE: Firmware information is available for vSphere 5.x/ESXi 5.x systems with the HP Offline Bundle or HP ESXi custom image installed only.

To view firmware information and/or update firmware for the selected server:

  1. Select a host in the inventory tree.

  2. Click the HP Insight Management tab.

  3. Select Software/Firmware.

    Detailed software and firmware information and status updates can be displayed.

Within the Software/Firmware page, you can do the following:

  • View software inventory

  • View firmware inventory

  • Generate a firmware inventory list in CSV file format by clicking Export (csv) at the bottom of the page

TIP: To update firmware, select Firmware from the Actions drop-down list.
