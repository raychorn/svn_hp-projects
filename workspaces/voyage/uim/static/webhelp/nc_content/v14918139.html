Internet Explorer script error occurs in environment with 100 or more LUNs

The following message occurs during a refresh in an environment with 100 or more LUNs:

Stop running this script? A script on this page is causing 
Internet Explorer to run slowly. If it continues to run, your computer might 
become unresponsive.

Possible cause: An Internet script is taking too long to complete.

Action: Click No.

Action: Add or modify the registry entry.

  1. Open the registry key HKEY_CURRENT_USER\Software\Microsoft\Internet Explorer\Styles. If the key does not exist, add it.

  2. Create a DWORD value called MaxScriptStatements under the key you opened or created in Step 1.

  3. Assign the value 0x00FFFFFFFF to prevent the script error. To prevent Internet Explorer from displaying this error message again, use the value 0xFFFFFFFF.
