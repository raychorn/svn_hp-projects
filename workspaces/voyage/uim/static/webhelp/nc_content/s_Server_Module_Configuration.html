Server Module Configuration

The Server Module Configuration page enables you to view and change the following parameters:

  • HP SIM port (default 50000)

  • Device poll interval (default 300)

  • Event poll interval (default 30)

  • Power cost (default 0)

  • Virtual Connect uplink over subscription factor (default 1.3)

To change a parameter, select it and then click the Edit icon .

Click the Refresh icon to initiate a manual cache refresh.
