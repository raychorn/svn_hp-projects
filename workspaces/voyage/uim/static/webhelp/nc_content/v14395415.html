No Virtual Connect information is available for this host error

The following scenarios can cause this error:

  • Servers are not blade servers.

  • The blade server is not configured with Virtual Connect.

  • The Virtual Connect credentials are not valid.

  • Other errors occurred while retrieving Virtual Connect data.

    For more information, see the Insight Control for vCenter logs. For information about accessing the logs, see Viewing the log files.
