Create Virtual Machines – Review and Validate

Review the information and validate the operation.

Pre-validation

  1. Review the information that will be used to create the virtual machine.

  2. If the information is correct, click Next to begin the validation.

    If the information is not correct, click Back and make the necessary changes.

    During the validation process a green icon indicates a successful step, and a red icon indicates failure. If a validation step fails, a message is displayed in the Warning and Validation Results box. Correct the problem, and then click Retry to repeat the validation step. If you need to change values entered in the wizard, use the Back and Next buttons to navigate.

Post-validation

  • When validation completes successfully, click Next to view a summary of the requested actions.
