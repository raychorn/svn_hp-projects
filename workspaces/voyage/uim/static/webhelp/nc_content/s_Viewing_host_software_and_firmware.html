Host software/firmware

The host software and firmware page includes details of the components installed and status updates.

Details view

This view identifies the firmware and software components installed on the host.

  • Name—The component on which the software or firmware is installed.

  • Description—The type of firmware or software.

  • Version—The firmware or software version installed.

Status Update view

This view includes the following job-related update information.

  • Job Description—A description of the job.

  • Status—The status of the job.

  • Status Description—Details about the job status.

  • Percent Complete—Indicates how much of the job has completed.

TIP: To update firmware, select Firmware from the Actions drop-down list.
