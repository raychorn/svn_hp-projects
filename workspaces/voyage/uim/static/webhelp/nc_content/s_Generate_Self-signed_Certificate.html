Certificate Management

This page enables you to manage certificates. You can perform the following tasks:

  • Generate a new self-signed certificate for Insight Control for vCenter.

  • Generate a certificate signing request to be signed by a Certificate Authority.

  • Install a signed certificate.

NOTE: After installing a self-signed certificate, you must restart all services for HP Insight Control for vCenter. After the services are started, log out of the vSphere Web Client, and then log back in.
