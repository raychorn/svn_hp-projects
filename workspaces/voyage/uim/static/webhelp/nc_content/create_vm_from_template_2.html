Create Virtual Machines – Specify Storage

Select the number of datastores to create for the host VMs.

  1. Enter the number of datastores to create to support the new VMs in Quantity.

    This value must be less than or equal to the number of new VMs. The VMs will be distributed among these datastores.

  2. Enter the minimum capacity for each new datastore in Datastore Capacity.

    The size of the datastore should be big enough to hold the VM(s). Use Per VM/Total capacity to calculate the size.

    NOTE: The datastore capacity value is approximate. The actual virtual disk sizes might be slightly larger to accommodate the VMware overhead.

  3. Select a storage pool to use for the new datastore in Storage Pools.

    The configured storage pools with Full Access on arrays that support provisioning are listed. If there is not enough space available in the selected storage pool, a message is displayed in the Warnings box.

    TIP: Select the Name or Available Capacity option to sort the list of storage pools.

  4. Select a RAID level for the new datastores in RAID Level.

    All RAID levels supported by the selected storage pool are listed.

  5. Click Next.
