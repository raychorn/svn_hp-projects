HP XP P9000 virtual disk information missing from Storage Overview page

Possible cause: The hostgroup name for the ports on the array exceeds 16 characters.

Action: Rename the hostgroup on the array to a name that is up to 16 characters, and then initiate a manual cache refresh. When the refresh process is complete, information will be displayed for the HP XP P9000 virtual disks.
