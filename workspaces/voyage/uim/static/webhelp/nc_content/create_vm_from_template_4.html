Create Virtual Machines – Specify Names

Specify names for the virtual machines and datastores being created.

  1. Enter the base name for the new VMs in VM Names.

    The Generated Names box displays the names of the VMs that will be created.

    The # symbol is required and represents the numeric sequence for the new names. In the generated VM names, the # symbol is replaced with a number.

    The default values are as follows:

    Number Default start value for numeric sequence
    1–9 1
    10–99 01
    100–999 001

  2. Enter the base name for the new datastores in Datastore Names.

    The # symbol is required and represents the numeric sequence for the new names. In the generated datastore names, the # symbol is replaced with a number.

    The Generated Names box shows the names of the datastores that will be created.

  3. For both the VM and datastore names, enter the start value for the numeric naming sequence in Where # is numeric and starts at.

    The default value is 1.

  4. Click Next.
