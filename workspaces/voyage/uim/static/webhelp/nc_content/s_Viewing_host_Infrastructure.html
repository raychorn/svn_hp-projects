Viewing host infrastructure information

The Host Infrastructure page displays information about blade status, enclosure status, and the enclosure power and thermal status as well as interconnect and OA information. The Infrastructure portlet, which displays an overview of the host infrastructure, is displayed on the Overview page.

NOTE: The Infrastructure portlet appears for blades only.

To view host infrastructure information:

  1. Select a host in the inventory tree.

  2. Click the HP Insight Management tab.

  3. Select Infrastructure .

    Detailed host infrastructure information is displayed.
