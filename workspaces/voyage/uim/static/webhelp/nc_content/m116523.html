Insight Control for vCenter events

Insight Control for vCenter loads events from Onboard Administrator, HP SNMP agents, and HP CIM providers into vCenter Server, enabling administrators to view and manage them from the vSphere Client. This provides administrators with a single, heterogeneous view of all host system events within the managed environment.

To view HP hardware tasks and events, navigate to the Tasks & Events tab in the vSphere Client, or the Events tab in the vSphere Web Client.
