Cluster information

Information is displayed for all hosts attached to the cluster selected.

To view details for the selected cluster:

  1. Select a cluster in the inventory tree.

  2. Click the HP Insight Management tab.

  3. Select Cluster.

    The cluster page opens listing details

    TIP: Use the expand icon at the beginning of each row to see details for the server/host.
