Accessing storage information on the Overview page

When using HP Insight Management with the Storage Module for vCenter, storage information appears in the News Feed, Software/Firmware, and Storage portlets, and in the consolidated status information for hosts and VMs.

  • Consolidated Status—Select a component in the inventory tree, and then move the mouse over the status icon to view the status summary.

  • Storage—Select a component in the inventory tree, and then click Storage to view details about the configured arrays and management servers.

    • Click the Summary tab for storage overview information, or click one of the following tabs for detailed information:

      • Storage Volumes

      • Virtual Disks

      • HBAs

      • Paths

      • Replications

      • VMs to Volumes

    • Click the Storage icon to access the available array management utilities for the configured arrays.

    • Click the cache refresh icon to refresh the Storage Module for vCenter information displayed in the vSphere Web Client. For more information, see Refreshing Storage Module for vCenter data.

    • Customize the tables as needed. For instructions, see Customizing Storage Module for vCenter tables.

    • Click the Help icon for context-sensitive help.

    For more information, see the following:

    • Viewing host storage information

    • Viewing cluster storage information

    • Viewing datastore storage information

    • Viewing VM storage information

    • Viewing detailed storage information

    • Managing hosts

    • Managing clusters

    • Managing datastores

    • Managing VMs

  • Software and firmware— To view software and firmware information, including read-only information about the storage associated with the selected component, click Software/Firmware.

    For more information about the Software/Firmware page, see Using the Server Module for vCenter.
