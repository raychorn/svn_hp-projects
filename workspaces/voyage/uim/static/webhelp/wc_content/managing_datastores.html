Managing datastores

To access the Overview page for a datastore, select a datastore in the inventory tree, then click Manage → HP Insight Management → Overview.

From the Overview page, you can do the following:

  • Access the News Feed, Storage, and Software/Firmware portlets. For more information, see Accessing storage information on the Overview page.

  • Perform provisioning operations from the Actions drop-down list. For more information, see ???.

  • View the News Feed. For more information, see Using the Server Module for vCenter.

  • View read-only information about the storage associated with the selected component. For more information, see Accessing storage information on the Overview page.
