Navigating HP Insight Management with the Server Module

NOTE TO REVIEWERS: Is SNMP supported in 7.3?

Information from iLO, Onboard Administrator, HP SNMP agents, HP CIM providers, and Virtual Connect Manager is compiled and displayed on the HP Insight Management page at both the cluster level and the host level.

NOTE: In order for iLO information to display, iLO must be associated with the VMware host. For more information, see Associating the VMware host to iLO.

If you have a server module installed and a cluster is selected in the inventory tree, HP Insight Management provides the following information:

  • Overview (Manage page)

  • Health (Monitor page)

  • Infrastructure (Manage page)

  • Software/Firmware (Manage page)

  • Networking

  • News Feed (Monitor page)

  • Cluster (Manage page)

  • Tasks (Monitor page)

If you have a server module installed and a host is selected in the inventory tree, HP Insight Management provides the following information:

  • Health (Monitor page)

  • Overview (Manage page)

  • Host (Manage page)

  • Networking (Manage page)

  • Software/Firmware (Manage page)

  • Infrastructure (Manage page)

  • Storage (Manage page) Displayed only if the storage module is installed.

  • News Feed (Monitor page)

  • Tasks (Monitor page)

NOTE: For information about navigating HP Insight Management with the Storage Module for vCenter, see Navigating HP Insight Management with the Storage Module.

Depending on the context of the page, links are available to launch HP tools, as needed.
