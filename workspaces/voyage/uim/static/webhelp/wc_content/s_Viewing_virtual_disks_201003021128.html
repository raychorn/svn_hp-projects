Viewing virtual disks

The virtual disks page displays the following information:

  • Virtual Disk Name—Virtual disk name assigned by VMware

  • Virtual Disk Type—Virtual disk type (VMDK or RDM)

  • Virtual Machine—Name of the VM associated with the selected VMDK or RDM

  • Datastore—Datastore name or the .vmdk name for the RDM

  • Provisioned Capacity—Disk capacity visible to a VM using this VMDK or RDM

  • Allocated Capacity—Disk capacity reserved by VMware for the VMDK or RDM

    For an RDM, this value will match the provisioned capacity value. For a VMDK, this value might be less than the provisioned capacity value if VMware thin provisioning is used. This is separate from thin provisioning configured in the storage array.

  • VMware Provisioned—Thick or thin
