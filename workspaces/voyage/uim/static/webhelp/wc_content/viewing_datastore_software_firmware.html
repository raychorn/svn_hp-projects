Viewing datastore software and firmware information

The datastore software and firmware page displays detailed information about the datastore software and firmware configuration. The Software/Firmware portlet on the Overview displays an overview of the software and firmware. Detailed software and firmware information is displayed on the Software/Firmware page.

To view firmware information and/or update firmware for the selected datastore:

  1. Select a datastore in the inventory tree.

  2. Click the Manage tab and then click HP Insight Management.

  3. Select Software/Firmware.

    Detailed software and firmware information is displayed.

Within the Software/Firmware page, you can do the following:

  • View software inventory

  • View firmware inventory

  • Generate a firmware inventory list in CSV file format by clicking Export (csv) at the bottom of the page
