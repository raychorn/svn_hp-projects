Cannot configure datastore block size

You cannot configure the block size for the VMFS-5 file system when creating a datastore in an vSphere 5.x/ESXi 5.x environment.

Cause: When creating datastores using the VMFS-5 file system, VMware does not support values other than 1 MB for the block size.
