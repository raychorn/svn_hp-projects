Launching into HP tools

Insight Control for vCenter provides links to launch HP tools if available. Some of the tools that can be launched from the HP Insight Management tab are as follows:

  • HP SIM

  • iLO

  • Onboard Administrator

  • Insight Control power management

  • Virtual Connect Enterprise Manager

  • HP storage array management utilities

  • QUIX issue QXCR1001299216

    HP OneView. This link is displayed if the selected host is managed by HP OneView. If a selected cluster contains hosts managed by HP OneView, this link will be displayed. If the cluster includes hosts managed by separate instances of HP OneView, mouse-over the OV icon to display a popup menu with links to each HP OneView instance.

When configured for vCenter administrators, auto-login is enabled for direct launch into the HP tools listed in this section with the exception of HP OneView and Virtual Connect Enterprise Manager, which do not support auto-login. Users with read-only access cannot access the HP tools. Other non-administrative vCenter Server users are prompted to enter a user name and password to access the HP tools.
