Internet Explorer Permission denied script error immediately after installation

The following message appears when you start the vSphere Client after installing Insight Control for vCenter:

Internet Explorer script error: Permission denied

Action: Close the window, add the vCenter host to the Internet Explorer list of trusted sites, and then start the vSphere Client again.
