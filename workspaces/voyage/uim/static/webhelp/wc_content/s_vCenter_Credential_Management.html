vCenter Credentials

This page lists the credentials for all vCenter server instances that are configured in the environment. You can perform the following tasks:

  • Add a new vCenter server instance—Click the Add icon and enter the credentials.

  • Edit a vCenter server instance—Select the appropriate server instance, click the Edit icon , and change the credentials.

  • Delete a vCenter server instance—Select the appropriate server instance and click the Delete icon .

  • Refresh—To initiate a manual cache refresh, click the refresh icon .

TIP:

  • To sort the information in a column, click on the corresponding column header. The up/down arrow to the right of the column header indicates the sort order.

  • Use the Filter tool () in the upper-right corner of the page to limit the information displayed to the specified criteria. Enter the data you want to filter on. You can limit filtering to specific columns by using the column selection sub-menu to select the desired columns. All columns are selected by default. All columns can also be deselected.

  • Use the Search tool () in the lower-left corner of the page to locate the specified data. You can limit searching to specific columns by using the column selection sub-menu to select the desired columns. All columns are selected by default. All columns can also be deselected.
