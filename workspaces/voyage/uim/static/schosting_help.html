Smart Component Hosting
HP Smart Components can be hosted by Insight Control for vCenter. Smart Component Hosting uses an HTTP web service which listens on port 63000.

The web service must be accessible by the ESXi hosts in order for component download to succeed. This means that the ESXi hosts must exist on the same management network as Insight Control for vCenter. If DNS names were used when configuring Insight Control, ensure that the ESXi hosts can resolve the hostname of the server where Insight Control is installed.

Port 63000 must also be open on the server where Insight Control for vCenter is installed. If any other services are using port 63000, Smart Component Hosting will not be able to start.
